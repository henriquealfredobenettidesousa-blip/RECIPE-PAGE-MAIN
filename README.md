### What I learned

This project helped me improve my understanding of semantic HTML and accessibility.

I learned how to structure content using headings based on hierarchy instead of visual appearance. I also learned how to use th with scope="row" to make table data more semantic and accessible.

It was also my first time working with a table and using multiple ordered and unordered lists in the same project.

On the CSS side, I practiced styling list markers, tables, and creating a responsive layout for different screen sizes.

### Continued development

In future projects, I want to continue improving my semantic HTML and accessibility knowledge.

I also want to improve how I handle responsive layouts between desktop and mobile breakpoints to create smoother transitions across different screen sizes.
